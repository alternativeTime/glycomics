Glycan Builder and GlycoDigest

An intuitive interface for building and displaying glycan structures

Welcome to the preview launch of UniCarbKBs integrated GlycoDigest tool. This first release focuses on the behaviour and mode of action of exoglycosidases on N-link glycans. To successfully run GlycoDigest build N-link structures with embedded linkage information
